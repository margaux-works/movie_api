API Documentation

The myFlix API is a RESTful service that provides access to a rich database of movie information, including details on movies, directors, and genres. It also offers user management features, enabling developers to integrate user registration, profile updates, and favorite movie lists into their applications.

URL Endpoints

Request URL HTTP Method Query Parameters Request Body Response
Get a list of ALL movies /movies GET n/a None A JSON array holding data about all movies
Get data about a single movie by its title /movies/:title GET title None A JSON object holding data about a single movie, including title, description, genre, director, and image URL.
Get movies by genre /movies/genre/:genreId GET genreId None A JSON array of movies that match the specified genre ID.
Get movies by director /movies/director/:directorId GET directorId None A JSON array of movies directed by the specified director ID.
Get data about a genre by ID /genres/:genreId GET genreId None A JSON object holding data about a genre, including genre name and description.
Get data about a director by ID /directors/:directorId GET directorId None A JSON object holding data about a director, including name, bio, birthdate, and date of death.
Register a new user /users POST n/a A JSON object with user details: { "Name": "Leo", "Email": "leodc@gmail.com", "Birthday": "YYYY-MM-DD", "FavoriteMovies": [] } A JSON object holding data about the registered user, including: Name, Email, Birthday, FavoriteMovies, and ID.
Update user's info /users/:userId PUT userId A JSON object with updated user details. A JSON object with the updated user details.
Add a movie to favorite list /users/:userId/favorites/:movieId PUT userId, movieId None A text message indicating the movie has been added to the favorite list.
`${movieId} has been added to user ${userId}'s favorite list`
Remove a movie from favorite list /users/:userId/favorites/:movieId DELETE userId, movieId None A text message indicating the movie has been removed from the favorite list.
`${movieId} has been removed from user ${userId}'s favorite list`
Deregister existing user /users/:userId DELETE userId None A text message indicating the user has been deregistered.
`User ${userId} has been deleted`
