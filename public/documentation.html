API Documentation

The myFlix API is a RESTful service that provides access to a rich database of movie information, including details on movies, directors, and genres. It also offers user management features, enabling developers to integrate user registration, profile updates, and favorite movie lists into their applications.

URL Endpoints

Request URL HTTP Method Query Parameters Request Body Response
Get a list of ALL movies /movies GET n/a None A JSON array holding data about all movies
Get data about a single movie by its title /movies/:Title GET Title None A JSON object holding data about a single movie, including title, description, genre, director, and image URL.
Get data about a genre by name /movies/genres/:Name GET Name None A JSON object holding data about a genre, including genre name and description.
Get data about a director by name /directors/:Name GET Name None A JSON object holding data about a director, including name, bio, birthdate, and date of death.
Register a new user /users POST n/a A JSON object with user details:
{ "Username": "Leo",
"Password": "password123",
"Email": "leodc@gmail.com",
"Birthday": "YYYY-MM-DD" } 		A JSON object holding data about the registered user, including: Username, Email, Birthday, FavoriteMovies, and ID.
Update user's info /users/:Username PUT Username A JSON object with updated user details:
{ "Username": "Leo",
"Password": "newpassword",
"Email": "leodc@gmail.com",
"Birthday": "YYYY-MM-DD" }
**Note:** The Username in the URL must match the currently authenticated user. 		A JSON object with the updated user details.
Add a movie to favorite list /users/:Username/movies/:MovieID PUT Username, MovieID None A JSON object of the updated user data, including the updated FavoriteMovies array.
Remove a movie from favorite list /users/:Username/movies/:MovieID DELETE id, MovieID None A message indicating that the movie has been removed from the user's favorites. `Movie with ID ${MovieID} was removed from user's favorites.`
Deregister existing user /users/:Username DELETE Username None A text message indicating the user has been deregistered.
`User ${Username} has been deleted`
